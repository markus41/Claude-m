# Template Catalog Strategy

## Goal

Standardize OneNote output quality through reusable templates.

## Required Template Fields

1. Template name
2. Version
3. Required sections
4. Required `data-id` anchors
5. Tag schema
6. Theme default

## Recommended Template Set

1. SOP template
2. Decision Record template
3. Incident Runbook template
4. Meeting Notes template
5. Weekly Status template

## Lifecycle

1. Draft template in sandbox section.
2. Validate XHTML compatibility.
3. Approve and version template.
4. Promote to production template library.
5. Retire old versions with migration notes.

## Quality Gates

1. One H1 and logical H2/H3 order.
2. Summary section required.
3. Action table schema present when applicable.
4. Tags present and normalized.
