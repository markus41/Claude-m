# TLD Comparison & Guide

## Primary TLDs

### .com (Commercial)
- **Status**: Gold standard for all businesses
- **Credibility**: Highest — users default to typing .com
- **Price**: $10-15/year registration, premium domains $500-$100K+
- **Best for**: Any business prioritizing mass-market trust
- **Drawback**: Extremely competitive — most short names taken
- **Tip**: If .com is unavailable, consider `get[name].com`, `[name]hq.com`, `use[name].com`

### .net (Network)
- **Status**: Strong secondary TLD, originally for ISPs/networking
- **Credibility**: High — well-established and trusted
- **Price**: $10-15/year
- **Best for**: Tech companies, SaaS platforms, network services
- **Drawback**: Some users may still type .com by habit
- **Tip**: Good alternative when .com is taken by a parked/inactive domain

### .org (Organization)
- **Status**: Traditional non-profit / community TLD
- **Credibility**: High for non-profits, moderate for commercial
- **Price**: $10-15/year
- **Best for**: Non-profits, open-source projects, community platforms
- **Drawback**: Implies non-commercial — may confuse for-profit users
- **Tip**: Excellent for open-source tools and community-driven projects

## Tech TLDs

### .io (Indian Ocean Territory)
- **Status**: De facto standard for developer/tech startups
- **Credibility**: High in tech, moderate elsewhere
- **Price**: $30-60/year
- **Best for**: Developer tools, APIs, SaaS, tech startups
- **Drawback**: Higher price, non-tech audiences may not recognize it
- **Examples**: GitHub.io, Socket.io, Notion.io (early)
- **Tip**: Widely accepted in Silicon Valley and tech communities

### .dev (Developer)
- **Status**: Google-backed TLD for developers
- **Credibility**: Growing — enforces HTTPS
- **Price**: $12-15/year
- **Best for**: Developer tools, coding platforms, tech portfolios
- **Drawback**: Newer TLD, less mainstream recognition
- **Examples**: web.dev, opensource.dev
- **Tip**: Great for developer-facing products; HTTPS requirement adds trust

### .app (Application)
- **Status**: Google-backed TLD for applications
- **Credibility**: Growing — enforces HTTPS
- **Price**: $15-20/year
- **Best for**: Mobile apps, web applications, software products
- **Drawback**: Newer TLD, some user unfamiliarity
- **Examples**: cash.app, get.app
- **Tip**: Perfect for app-first businesses; HTTPS-only adds security signal

### .ai (Artificial Intelligence)
- **Status**: Booming TLD for AI companies (actually Anguilla ccTLD)
- **Credibility**: High for AI/ML companies
- **Price**: $50-100/year
- **Best for**: AI startups, machine learning platforms, data companies
- **Drawback**: Expensive, implies AI focus
- **Examples**: stability.ai, character.ai, perplexity.ai
- **Tip**: Premium pricing but strong signal for AI-focused companies

### .co (Colombia / Company)
- **Status**: Popular startup alternative to .com
- **Credibility**: Moderate to high
- **Price**: $25-35/year
- **Best for**: Startups that can't get .com, global brands
- **Drawback**: Can be confused with .com typos
- **Examples**: angel.co, Twitter's t.co
- **Tip**: Works well alongside a .com redirect strategy

## TLD Selection Decision Tree

```
Is your audience primarily developers/tech?
├── Yes → .io, .dev, .app, .ai (depending on product type)
└── No → Is .com available?
    ├── Yes → Use .com (always preferred for non-tech)
    └── No → Is your business a non-profit?
        ├── Yes → .org
        └── No → Try .co, .net, or creative .com variants
```

## Multi-TLD Strategy

For serious brands, consider registering multiple TLDs:

1. **Primary domain** — Main website (e.g., `brandname.com`)
2. **Redirect domains** — Common alternatives redirect to primary
3. **Defensive registrations** — Prevent competitors/squatters
4. **Country-specific** — For international expansion

**Minimum recommended:**
- Register `.com` + your primary TLD
- Set up redirects from alternatives
- Monitor expiration dates
